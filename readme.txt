Plannedsoft Min/Max Quantities is the best Woocommece add-on to restrict users to buy products with restriction for minimum and maximum quantities.

== Description ==

**Plannedsoft Min/Max Quantities - Best Woocommerce Add-on**
★★★★★<br>

See the features which are exclusive to the Plannedsoft Min/Max Quantities**.

* **Minimum/Max Order**
Set maximum order value to either limit the amount a user can spend. Or set a minimum order value a user spend to ensure every order remains profitable.

* **Minimum/Max Quantity**
There will be times when you want to Set minimum, maximum, quantity rules.

* **Default Quantity**
Set quanity defualts on a global or product level to make managing your site effortless.

* **Quantity Increments**
Create order increments on all or selected products that you want to sell in batches.

* **Easy Setup**
Get setup in minutes with our plugin and play system.

== Installation ==

1. Visit `Plugins > Add New`
2. Search for `Plannedsoft Min/Max Quantities`
3. Install Plannedsoft Min/Max Quantities once it appears
4. Activate Plannedsoft Min/Max Quantities from your Plugins page.
5. Go to "after activation" below.

= Manually =

1. Upload the `plannedsoft-min-max-quantities` folder to the `/wp-content/plugins/` directory
2. Activate the Plannedsoft Min/Max Quantities plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

= After activation =

1. You should see the Plannedsoft Min/Max Quantities setup wizard.
2. Go through the setup wizard and set up the plugin for your site.
3. You're done!
